# Imagination--WhiteBoard-Draw-your-Imagination, Online Whiteboard Application

An advanced Online Whiteboard Web App built using HTML, CSS, and Vanilla JavaScript that allows users to draw, write, design, and manage content on an infinite-style canvas just like a real digital whiteboard.

Live Link: 

https://imagination-drawing.netlify.app/

🚀 Features
✏️ Drawing Tools

Pencil Tool with:

Adjustable size (stroke width)

Multiple brush types:

Pen

Marker

Highlighter

Smooth stroke rendering

Eraser Tool

Erase parts of pencil strokes

Erase shapes by touching their borders

Drag-to-erase support

🧱 Shapes

✅ Draw Rectangles

✅ Draw Circles / Ellipses

✅ Draw Lines

✅ All shapes support:

Move / drag

Delete using eraser

📝 Text Tool

Add text anywhere on canvas

Edit existing text by clicking on it

Cursor placement inside text

Supports:

Backspace / Delete

Arrow navigation

Live editing

Move text using Move tool

✋ Move Tool

Drag and move:

Rectangles

Circles

Lines

Text

Works exactly like selection tool in real whiteboard apps

🎨 Colors & Styling

Color picker for:

Pencil

Shapes

Text

Adjustable pencil thickness

Different brush effects (marker, highlighter, pen)

🗑️ Eraser System

Erase:

Pencil strokes (partial erase supported)

Rectangles (by border touch)

Circles

Lines

Smart stroke splitting while erasing

↩️ Undo / Redo

Full Undo (Ctrl + Z) support

Full Redo (Ctrl + Y / Ctrl + Shift + Z) support

State-based history management

Works for:

Drawing

Moving

Deleting

Editing

📄 Export

Export entire board as:

PDF

(Canvas-based export)

🧭 Infinite Board Feel

Large 3000 × 3000 canvas

Scrollable in all directions

Auto-centered on load

Works like a real whiteboard workspace

🖱️ UX Features

Custom cursors for:

Pencil

Move

Text

Eraser

Toolbar with toggle states

Responsive UI (works on mobile screens too)

🛠️ Tech Stack

HTML5

CSS3

JavaScript (Vanilla)

Canvas API

jsPDF (for PDF export)
